## 🚀 Como Executar

### Pré-requisitos
- Node.js v16+
- npm ou yarn

### Instalação
```bash
git clone https://github.com/seu-usuario/portfolio-react.git
cd portfolio-react
npm install
npm run dev
```
### Acesse

http://localhost:5173
